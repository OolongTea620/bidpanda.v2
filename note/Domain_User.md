# User 도메인 명세

## 비즈니스 요구 사항

```markdown
회원가입

- email가 다르면 다른 회원으로 정의한다.
- 로그인(인증,인가)를 위한 password가 있어야한다.
- 닉네임이 있어야 한다. (중복 가능하다)
- 프로필 url을 필요로 한다. (없으면 기본 url을 사용한다)
- 탈퇴한 이메일은 재가입이 불가능하다.

회원 정보 수정

- 자신의 정보만 수정이 가능하다.
- 닉네임 수정이 가능하다.
- 비활성화된 회원의 정보는 수정이 불가능하다.

회원 탈퇴(회원 비활성화)

- 이미 탈퇴한 회원은 탈퇴가 불가능하다.
- 회원은 자기 자신을 탈퇴요청 할 수 있다.
- 탈퇴한 회원 == 비활성화 회원이다.
```
